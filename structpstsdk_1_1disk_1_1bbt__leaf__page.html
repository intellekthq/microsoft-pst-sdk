Detailed Description

template<typename T>
struct pstsdk::disk::bbt_leaf_page< T >

BBT leaf page.

A BTree page instance. The BBT leaf page has an array of bbt_leaf_entries ordered by block id, which describe the blocks in the database.

Template Parameters
Tulonglong for a unicode store, ulong for an ANSI store
